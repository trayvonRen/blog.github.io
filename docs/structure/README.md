在计算机科学中，数据结构（英语：data structure）是计算机中存储、组织数据的方式。

## 数据结构的分类
<img src="/img/dt.png" width = "60%" height = "60%" align=center />     

- 线性结构  
   - 线性表(Linear list)
      - 顺序存储结构：顺序表  
      - 链式存储结构：链表(link list)
   - 队列  
   - 栈  
   - 串(string)
   - 数组  
- 非线性结构  
   - 树   
      - 二叉树  
         - 二叉搜索树

- 集合 
   - 键值对 
      - map 
         - hashMap    
      - hashTable  
      - dictionary
