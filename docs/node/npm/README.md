## npm 是什么？
npm（全称 Node Package Manager，即“node包管理器”）是Node.js默认的、以JavaScript编写的软件包管理系统。

npm 由三个独立的部分组成：

- 网站  
- 注册表（registry）  
- 命令行工具 (CLI)  

网站 是开发者查找包（package）、设置参数以及管理 npm 使用体验的主要途径。  
注册表 是一个巨大的数据库，保存了每个包（package）的信息。  
CLI 通过命令行或终端运行。开发者通过 CLI 与 npm 打交道。
