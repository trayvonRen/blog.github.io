## react & react-dom

创建一个 react 项目时，依赖中会出现两个包：react react-dom  
react 包是 React 无关平台的核心逻辑。例如组件实现、更新调度等。  
react-dom 包则是 React 在 web 平台上负责操作 dom 的逻辑。例如 ReactDOM.render() 和 ReactDOM.findDOMNode()。
